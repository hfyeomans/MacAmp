# Research: Playlist List Operations

> **Purpose:** Research Winamp playlist LIST OPTS button behavior, M3U/PLS format support, and existing MacAmp M3U parsing capabilities.

**Status:** Pending research/planning.

---

## Context

Winamp's playlist window has buttons for NEW LIST (clear playlist and start fresh), LOAD LIST (load .m3u/.pls playlist file), and SAVE LIST (export current playlist as .m3u). MacAmp has M3U parsing support but no UI buttons for these operations. The buttons appear in the playlist window bottom bar.
