# Research: Stream Track Counter

> **Purpose:** Research Winamp track counter display behavior and sprite/layout requirements for main player and playlist windows.

**Status:** Pending research/planning.

---

## Context

Winamp shows a track counter (e.g., "3/15") in the main player window and playlist window. MacAmp needs this for both local playlists and streaming stations. The main window counter shows current track / total tracks. The playlist window counter shows the count in the playlist header area.
